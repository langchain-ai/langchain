# v1.1_改动1_.profile.md

## 1. 如何在代码中使用 .profile

```py
from langchain import init_chat_model  # 举例

# 使用了某个模型后， 使用.profile能够拿到这个模型的属性
model = init_chat_model("some-model-id")
print(model.profile)
# 可能输出类似:
# {
#   "max_input_tokens": 400000,
#   "image_inputs": True,
#   "reasoning_output": True,
#   "tool_calling": True,   
#   "structured_output": True, // 是否支持结构化输出
#   ...
# }

```

如果发现 profile 数据不准确，还可以：

```py
custom = {
  "max_input_tokens": 100_000,
  "tool_calling": True,
  "structured_output": True,
  # … other keys …
}
model = init_chat_model("your-model-id", profile=custom)

# 或者，对已有 model 实例做 copy + 修改：

new_profile = model.profile | {"some_key": some_value}
model2 = model.model_copy(update={"profile": new_profile})

```
