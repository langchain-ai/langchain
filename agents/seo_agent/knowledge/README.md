# Ralf's SEO & AEO Knowledge Base

This directory contains Ralf's reference knowledge on SEO and AEO best practices.
Files are loaded on-demand when Ralf needs to make decisions about strategy, content,
or technical implementation.

## Structure
- `seo_fundamentals.md` — Core SEO principles that don't change
- `technical_seo.md` — Crawlability, indexing, Core Web Vitals, structured data
- `content_seo.md` — Content strategy, E-E-A-T, topical authority, content clusters
- `link_building.md` — Modern link building tactics, what works in 2026
- `local_seo.md` — Local SEO for UK-focused sites
- `aeo.md` — AI Engine Optimisation: optimising for AI search (Perplexity, ChatGPT, Gemini)
- `seo_2026_trends.md` — Current trends and algorithm updates
- `tools_and_apis.md` — How to use Ahrefs, GSC, and other tools effectively

## How Ralf Uses This
The knowledge base is injected into the system prompt when Ralf needs to make
strategic SEO decisions. It's also used by the content_writer node to ensure
blog posts follow current best practices.

## Keeping It Updated
The heartbeat periodically searches for new SEO/AEO developments and updates
these files. Last comprehensive update should be noted at the top of each file.
